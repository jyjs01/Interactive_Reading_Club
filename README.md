# Interactive Reading Club

사용자들이 책을 함께 읽고 토론할 수 있는 온라인 독서 클럽을 제공하는 웹 애플리케이션입니다.
이 프로젝트는 React를 기반으로 개발되었습니다.

## 실행 방법

프로젝트 폴더에서 명령 터미널을 통해 실행합니다.


## 클라이언트 (port : 3000)

### `npm start` 

프론트엔드 코드를 실행시킵니다.


## 서버 (port : 4000)

### `cd backend`

또 다른 명령 터미널을 열어 backend 폴더로 이동 후 

### `node server.js`

서버를 실행시킵니다.
